# Lost & Found Manager

## Problem
Students often lose items on campus, and there is no efficient system to recover them.

## Solution
This project allows users to:
- Report lost items
- Report found items
- Match items based on similarity

## Features
- Simple CLI interface
- Offline usage
- Basic matching logic

## How to Run
1. Clone the repository
2. Run:
   python main.py

## Example
Report lost item → Someone reports found → System suggests match

## Learning Outcomes
- Data storage using JSON
- Basic matching logic
- Modular programming
